# Search Highlight App

This project is a simple React application that allows users to search through a list of articles and highlights matching keywords in real time.

## Installation

1. Clone the repository:

git clone https://github.com/your-username/search.engine

2. Navigate into the project folder:

cd search-highlight-app

3. Install dependencies:

npm install

4. Run the app:

npm start

The app will run at:
http://localhost:3000
